Project Summary

This project establishes a complete data pipeline to extract, clean, analyze, and visualize public product data from an e-commerce platform (Banggood). The goal is to provide actionable business intelligence on product performance, pricing strategy, and customer satisfaction using key metrics like Value Score and Popularity Index.

 Automated Data Extraction: Python-based scraping solution to collect real-time product listings.

Data Quality & Enrichment: Comprehensive cleaning process, including handling missing data and calculating derived features.

Centralized Storage: Data persistence and efficient querying via a SQL Server database.

Advanced Analytics: SQL queries and Python analysis to derive Key Performance Indicators (KPIs) and business insights.

Interactive Dashboard: A Streamlit web application for real-time visualization and filtering of product metrics.
